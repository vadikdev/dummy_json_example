# dummyjson.com api integration example

## Dependencies
* node.js
* React.js

## Install back-end dependencies & node.js start server:

```
npm install
npm start

```

## Install back-end dependencies & run front-end React.js application:

```
cd ./client
npm install
npm start
```
## Open front-end application:

```
127.0.0.1:3000
```
