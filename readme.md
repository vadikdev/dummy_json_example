# dummyjson.com api integration example

## Dependencies
* node.js
* React.js

## Install dependencies:

```
npm install
```

## Start back-end node.js server:

```
npm start
```

## Run front-end React.js application:

```
cd ./client
npm start
```
## Open front-end application:

```
127.0.0.1:3000
```
